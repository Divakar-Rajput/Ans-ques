Computer Architecture Quiz

1. ____ addressing mode is most suitable to change the normal sequence of execution of instructions.

Answer : A) Relative

Explanation : The relative addressing mode is used for branching and can alter the normal execution flow by specifying an address relative to the current instruction.

2. Which of the following is not a valid addressing mode?

Answer: D) Floating-Point

Explanation : Floating-Point is not an addressing mode but rather a data type. Register, Absolute, and Immediate are valid addressing modes.

3. _____ bus structure is usually used to connect I/O devices.

Answer: A) Single bus

Explanation : BUS is a bunch of wires that carry address, control signals, and data. It is used to connect various components of the computer.

4. The I/O interface required to connect the I/O device to the bus consists of _____

Answer: C) Address decoder, registers, and control circuits

Explanation : The I/O devices are connected to the CPU via BUS, and to interact with the BUS, they have an interface.

5. To reduce the memory access time, we generally make use of _____

Answer: D) Cache's

Explanation : Cache memory is used to store frequently accessed data, reducing memory access time.

6. Which of the following is not a valid addressing mode?

Answer: D) Floating-Point

Explanation : Floating-Point is not an addressing mode but rather a data type. Register, Absolute, and Immediate are valid addressing modes.

7. Which type of memory is non-volatile and retains data even when power is off?

Answer: B) ROM

Explanation : ROM (Read-Only Memory) is non-volatile and retains data even when the power is off.

8. What does the ALU (Arithmetic Logic Unit) of a computer perform?

Answer: C) Arithmetic and logic operations

Explanation : The ALU performs all arithmetic and logical operations in a computer.
